# Assets

This MVP uses temporary code-generated assets only.

## Generated In Code

- Player, enemy, boss, chest, projectile, slash, AoE, warning, and item card textures are generated in `src/game/scenes/BootScene.ts`.
- Sound effects are generated with WebAudio oscillators in `src/game/systems/SoundSystem.ts`.

## External Assets

No external game art, sound effects, fonts, or other third-party assets have been added yet.
